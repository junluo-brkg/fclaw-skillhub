# FClaw Skill Hub

Official skill repository for FClaw on FlowOS.

Browse, download, and install skills for Foxx devices.

## About

FClaw Skill Hub is the central repository for distributing skills compatible with FClaw, the built-in AI assistant of FlowOS.

This repository provides skill packages, metadata, release information, and installation resources for both official and community-developed skills.

## Features

- Official FClaw skills
- Community-contributed skills
- Versioned releases
- Secure GitHub distribution
- FlowOS compatibility information

## Installation

Skills can be downloaded from the Releases section or installed directly through supported versions of FClaw.

1. Browse available skills
2. Download the desired skill package
3. Import or install the package in FClaw
4. Start using the new capability
